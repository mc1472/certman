# Certman
A utility for creating and manage ssl Certificates.

```
Usage: certman [OPTIONS] <COMMAND>

Commands:
  ca
  certs
  help   Print this message or the help of the given subcommand(s)

Options:
  -c, --config <CONFIG>      the config file
      --non-interactive
  -f, --force
      --dn-file <DN_FILE>
      --san-file <SAN_FILE>
  -h, --help                 Print help
  -V, --version              Print version
```
